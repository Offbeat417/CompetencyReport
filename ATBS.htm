Introduction to Python

Python is programming language that has been on the rise in the past couple decades becoming only second to JavaScript, which surprisingly enough has started to run for its money. Because of its simplistic design and ease of learning, learning programming has become a far more obtainable goal than ever before. While learning Python has indeed been made easier, it can still be considered a daunting task by most. Automate the Boring Stuff With Python has been created to help aspring coders to understand the basic fundamentals of Python, and how big a difference you can make in life by simply knowing a mere amount of it.



Order of Operations

Operator ** == Exponent

Operator % == Modulus/Remainder

Operator // == Integer division/flooded quotient

Operator / == Division

Operator * == Multiplication

Operator - == Subtraction

Operator + == Addition



Common Data Types

Data Type: Integers == -1 -2 0 1 2 3 4 5(Whole real and imaginary numbers)

Data Type: Floating-point numbers == -1.25 -1.0 -.5 0.0 0.5 1.0 1.25 (In other words decimals)

Data Type: Strings == 'a' 'aa' 'aaa' 'Hello!' '11 cats' (basically, words and what not)
